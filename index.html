Hexo + Github 搭建个人博客

  前言：心血来潮，想开始写blog，来记录整理自己所学所闻。经同事介绍利用Hexo+Github的博客系统形式来进行管理书写，但在配置上遇到不少坑，所以总结下。

  其实网上现成的教程攻略特别多，所以推荐两份较为详细的攻略来参考，再总结自己遇到的问题

https://www.jianshu.com/p/834d7cc0668d 里面搭建的每个步骤都写的十分详细了

总结问题：

  1.配置Github的ssh key的时候，如果使用的电脑正好有公司的相关key，记得分清楚，不要混淆

  2.再配置Hexo的_config.yml文件时，主要脚本语言的书写规范（特别执行的命令冒号后要记得空一行！！！）

  3.域名购买的选择，如果不嫌备案麻烦可以买国内一些知名的代理商旗下的，如果嫌麻烦，那老老实实用Godaddy去买，挺方便（吐槽下：这个网站交互相当不友好！！！），然后利用DNSPod去设置dns配合使用就基本ok啦！
